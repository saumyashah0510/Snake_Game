# Snake Game (C++)

## Description:
This is a terminal based snake game implemented in C++. It features different levels and runs on both Windows and Linux/MacOS.

## Features:

- OS Independent:
    - Works on both Windows and Linux\MacOS.

- Multiple Game Modes:
    - Easy (Slow Speed)
    - Medium (Medium Speed)
    - Fast (Fast Speed and obstacles)
